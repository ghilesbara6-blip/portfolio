    Bara Ghiles Web & Cross-Platform Developer · Béjaïa

    I go end to end, and I go deep. Based in Béjaïa, I build web platforms and cross-platform mobile apps — React, Flutter, vanilla JS — with backends in whatever fits the problem. I'm comfortable at every layer of the stack, from the browser to the kernel.

    My background runs through reverse engineering, OS-level programming in C and C++, and offensive cybersecurity — so when I build something, I build it knowing how it can be broken. I use the terminal as my main workspace, whether that's PowerShell, CMD, or Bash, and I treat research as part of the job.

    Right now I'm exploring local AI model development — running, fine-tuning, and eventually building inference systems that don't need a cloud. If I don't know how something works, I open it up, read it, and figure it out. That's not a workaround. That's the method.
